新手入門 / Base

5分鐘法式微光偽素顏妝容

如何使用最少的美妝步驟，打造極致無瑕、透亮乾淨的微光感妝容，適合日常通勤與約會。

步驟解析：

  1. 基礎保濕：上妝前使用保濕精華，待肌膚完全吸收。
  2. 點塗粉底：取半泵【絲絨光感粉底液】，輕點於面部中心區域。
  3. 均勻拍開：使用濕潤的美妝蛋由內向外拍開，保留兩頰天然光澤。
  4. 局部遮瑕：使用【無瑕無痕遮瑕膏】輕點於黑眼圈與暗沉處。
  5. 定妝提亮：T區輕掃【柔焦控油蜜粉】，蘋果肌點塗【極致星光高光液】。
眼妝進階 / Eyes

大地之光漸層立體眼妝技巧

深入解析九色眼影盤的漸層搭配，從亮片打底到深邃眼尾，打造明亮有神的深邃電眼。

步驟解析：

  1. 大面積打底：使用霧面米色鋪滿整個眼皮，均勻眼周膚色。
  2. 眼褶加深：用暖棕色暈染在雙眼皮褶皺內，並向外眼角自然過渡。
  3. 眼尾描繪：深褐棕色點塗在眼尾 V 字區，向內暈染，增加立體感。
  4. 眼影提亮：指腹蘸取金色亮片，點在眼皮正中央與臥蠶處。
  5. 線條勾勒：使用【極細極黑眼線筆】順著眼形描繪細緻眼線。
日常技巧 / Tips

完美持妝：如何使用柔焦控油蜜粉

解鎖專業彩妝師最愛的「烘焙定妝法」與「局部滾動法」，確保妝容 12 小時不脫妝、不浮粉。

步驟解析：

  1. 折疊按壓：將蜜粉倒在粉撲上，對折揉搓使粉末分布均勻。
  2. 滾動手法：在容易出油的T字部位，用粉撲輕輕滾動按壓定妝。
  3. 烘焙手法：在眼下遮瑕處敷上一層較厚蜜粉，停留 2 分鐘後用餘粉刷掃淨。
  4. 噴霧加乘：定妝完噴灑保濕噴霧，使粉體完美融入肌膚，呈自然柔霧感。
唇妝描繪 / Lips

豐盈飽滿法式雙唇與暈染技巧

如何使用霧面絲絨唇膏畫出邊界模糊、性感豐盈的咬唇妝，與俐落精緻的高級滿唇妝。

步驟解析：

  1. 唇部打底：上妝前 10 分鐘塗抹潤唇膏，並在上妝前用面紙吸乾。
  2. 內側塗抹：將【霧面絲絨唇膏】點塗在嘴唇內側三分之二處。
  3. 棉花棒暈染：使用棉花棒或指腹將唇膏邊緣向外暈染，淡化明顯唇線。
  4. 唇峰提亮：取極少量高光液輕點在人中V字與唇峰上邊緣，視覺倍增豐滿。
